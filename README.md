# Graduation-Project
- IoT 장치의 제어 신호를 스니핑하여 정상 작동여부를 확인하고 log 데이터로 남길 수 있도록 하는 GUI 기반의 모니터링 프로그램 제작
- IoT <-> 허브, IoT <-> Phone, Cloud <-> Phone, Cloud <-> HUB 간의 제어신호 송,수신 파악

# Development Environment
- Windows10(64bit)
- Python3.6
- Flask 기반
- Source tree 기반 협업
